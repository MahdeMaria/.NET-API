# Cadastro de Usuário

# Objetivo
Este projeto é uma versão aprimorada do meu primeiro cadastro de usuario usando C# e .NET nessa versão temos o CRUD e documentação usando o swagger.

# Estrutura do Projeto

Acões:
- Listar Usuario
- Criar Usuario 
- Editar Usuario
- Deletar Usuario

# Como rodar:

1 - Buildar:
```bash
dotnet build
```
2 - Executar:
```bash
dotnet build
```
